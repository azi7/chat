Chat Application using WebSocket

This project implements a simple chat application using WebSocket technology. The application consists of a WebSocket server (implemented in Node.js) and a client-side interface (HTML and JavaScript) that allows users to send and receive messages in real-time.

Project Structure

- `server.js`: Contains the WebSocket server implementation using Node.js.
- `index.html`: Client-side interface allowing users to send and receive messages.

## How to Run

1. Install Dependencies:
   Make sure you have Node.js installed. If not, you can download it from [nodejs.org](https://nodejs.org/).

2. Clone the Repository:
   
   git clone <repository-url>
   cd chat-application-websocket
   

3. Install Packages:
   
   npm install
   

4. Run the Server:
   
   node server.js
   

5. Open the Client:
   Open `index.html` in your web browser. You can send and receive messages through the client interface.

Dependencies

- `ws`: WebSocket library for Node.js.
